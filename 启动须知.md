# Whispers of the Forgotten Realm — 启动须知

## 前置条件

| 依赖 | 版本要求 | 检查命令 |
|------|----------|----------|
| Node.js | ≥ 18 | `node -v` |
| MySQL | ≥ 8.0 | `mysql -u root -p` |
| npm | ≥ 9 | `npm -v` |

MySQL 需提前创建好用户并确保服务正在运行。默认连接配置：

```
Host:     127.0.0.1
Port:     3306
User:     root
Password: 123456
Database: rpg_game（自动创建）
```

如需修改，编辑 `server/config/index.js` 中的 `DB_CONFIG`。

---

## 首次启动

```bash
# 1. 安装依赖（仅首次）
npm install

# 2. 启动 API 服务（终端 1）
node server/index.js

# 3. 启动前端（终端 2）
npm run dev
```

API 服务启动后会自动创建数据库和表，并写入预设管理员账号。

浏览器打开 `http://localhost:5173`。

## 公网访问（ngrok）

### 安装 ngrok

1. 访问 [ngrok.com](https://ngrok.com) 注册免费账号
2. 下载 ngrok 客户端并解压
3. 运行一次认证：`ngrok config add-authtoken <你的token>`

### 启动隧道

```bash
# 方式一：Python 脚本（Windows/Linux/macOS 通用）
python ngrok_public.py

# 方式二：直接运行 ngrok
ngrok http 5173
```

成功后显示：

```
Forwarding  https://xxxxx.ngrok-free.app -> http://localhost:5173
```

把 `https://xxxxx.ngrok-free.app` 发给其他人，手机或电脑浏览器打开即可。

### 常见问题

**报错 "ERR_NGROK_334 / endpoint is already online"**

说明之前的 ngrok 隧道没断开。先杀掉旧进程再重试：

```bash
# Windows
taskkill /F /IM ngrok.exe

# macOS / Linux
pkill ngrok
```

等待 30 秒后重新启动隧道。

### 注意事项

- 免费版 ngrok 只能同时开一个隧道
- 免费版有访问频率限制，多人同时访问可能被限流
- 每次重启 ngrok 链接会变化
- ngrok 自带 HTTPS 证书，无需额外配置
